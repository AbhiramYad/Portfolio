# My Portfolio Website

A simple, modern portfolio website built with HTML and CSS.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Easy to Customize**: Simple HTML and CSS structure
- **Sections**: Hero, About, Projects, Skills, and Contact

## Project Structure

```
├── portfolio.html      # Main HTML file
├── portfolio.css       # CSS styling
└── README.md          # Documentation
```

## How to Use

1. Clone this repository or download the files
2. Open `portfolio.html` in your web browser
3. Customize the content with your own information
4. Replace project descriptions and skills with yours
5. Update contact information as needed

## Customization

### Update Personal Information
- Edit the hero section heading and description
- Update the about me section
- Add your actual projects
- List your real skills and proficiency levels

### Change Colors
Edit the CSS variables in `portfolio.css`:
```css
:root {
    --primary-color: #0066cc;      /* Main color */
    --secondary-color: #f39c12;    /* Accent color */
    --text-color: #333;
    --light-bg: #f8f9fa;
}
```

### Add Social Links
Update the social links in the footer with your GitHub, LinkedIn, and other profiles.
